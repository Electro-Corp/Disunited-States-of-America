# Nunticle
A real time stratagy game about naval combat. <br>

## Pre-Reqs
* a modern-ish c++ compiler that supports "#pramga"
* SFML
* Lua 5.1
* LuaBridge

## Build
To build, create a empty build directory:
```
mkdir build
```
Then, go into it and run cmake:
```
cd build
camke ../
```
Then, compile and run!
```
make
./nunticle
```
